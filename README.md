# 🏥 Medical Insurance Charges Predictor

## 📌 Project Overview
The **Medical Insurance Charges Predictor** is a Machine Learning web application that predicts the estimated medical insurance cost based on a person's health and lifestyle information.

The prediction is made using important factors such as **Age, BMI, and Smoking Status**. This project demonstrates how machine learning can be used to assist in estimating healthcare expenses.

---

## 🚀 Features
- Predict medical insurance charges
- Simple and interactive user interface
- Real-time prediction using Machine Learning
- User inputs:
  - Age
  - BMI
  - Smoking status
- Instant prediction result

---

## 🧠 Machine Learning Model
The model was trained using a **Medical Insurance Dataset**.

### Input Features
- **Age** – Age of the person
- **BMI** – Body Mass Index
- **Smoking Status** – Whether the person smokes or not

### Output
- **Predicted Medical Insurance Charges**
